Decentralized Blockchain Network

A decentralized blockchain project built using Python and Flask.

Features
Proof of Work Mining
Peer-to-Peer Node Communication
Blockchain Synchronization
Automatic Mining
Wallet Balance Management
ECC & ECDSA Security
SHA-256 Hashing
Offline Node Recovery
REST API Integration
Technologies Used
Python
Flask
Flask-CORS
SHA-256
ECC Cryptography
ECDSA Digital Signatures
JSON Storage
Threading
Run the Project
Start Node 5000
python blockchain.py -p 5000
Start Node 5001
python blockchain.py -p 5001
Start Node 5002
python blockchain.py -p 5002

Project Goal

This project demonstrates the core concepts of blockchain including:

mining
consensus
synchronization
transaction verification
decentralized networking

Inspired by blockchain systems like Bitcoin and Ethereum.
